## Overview

Architected and deployed FreeWheel's first end-to-end data pipeline
for the APAC region, integrating automated ETL processes with
interactive PowerBI and Looker dashboards for executive reporting.

## Context

FreeWheel is a Comcast company providing advertising technology
solutions. When I joined as a Data Analyst Intern, the APAC region
had no centralised data infrastructure — reporting was done manually
with significant duplication of effort across teams.

## What I Built

**ETL Pipeline**
Designed and implemented scalable ETL processes using Python and Excel
to automate large-scale data extraction, cleaning, and transformation
from multiple source systems. The pipeline reduced manual processing
time by 75%.

**Executive Dashboards**
Built interactive dashboards in both PowerBI and Looker to surface
key APAC marketing metrics for executive leadership. Dashboards are
refreshed automatically on the pipeline schedule.

**Data Architecture**
Designed the overall data architecture to be extensible — new data
sources can be integrated into the pipeline without restructuring
the existing flow.

## Impact

- 75% reduction in manual data processing time
- First centralised data infrastructure for APAC region
- Executive leadership now has real-time visibility into APAC metrics
- Standardised reporting across International Marketing team

## Key Takeaways

- Building data infrastructure from scratch requires as much
  stakeholder management as technical work
- Automation is most valuable when it removes repetitive work from
  people who have higher-value things to do
- Dashboard design is as important as the underlying data — if
  executives don't understand it, it won't get used
